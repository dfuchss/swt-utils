# Maven & Co.
If you want to use maven or some similar tool add the following code to your pom:
```xml
<dependencies>
    <dependency>
        <groupId>org.fuchss</groupId>
        <artifactId>swt-utils</artifactId>
        <version>X.Y.Z</version>
    </dependency>
</dependencies>
```
